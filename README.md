# Solid Principles Project

### This project will help me learn the SOLID pattern.

# Get started

1. Install all dependencies
```bash
 yarn
 ```

 2. Install husky hooks
 ```bash
yarn husky install
 ```

 3. Create a .env file and copy all data of the .env.example file

 4. Start server
 ```bash
 yarn dev
 ```

 5. Run all tests
 ```bash
 yarn test
 ```
